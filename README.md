
# 🏥 QMC Hospital Management System

A console-based Hospital Management System developed in C++ using Object-Oriented Programming (OOP) concepts. This project manages patients, doctors, staff, and appointments with file handling support for permanent data storage.

---

## 🚀 Features

### 👨‍⚕️ Patient Management
- Add new patients
- View all patients
- Search patient by ID
- Update patient details
- Delete patient records

### 🩺 Doctor Management
- Add doctors
- View all doctors

### 👨‍💼 Staff Management
- Add staff members
- View all staff

### 📅 Appointment Management
- Schedule appointments
- View appointment records

### 📊 Reports
- Total patients
- Total doctors
- Total staff
- Total appointments
- Admitted patient statistics

### 💾 File Handling
- Automatic data saving and loading using text files

---
